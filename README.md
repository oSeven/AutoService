Kotlin AutoService

# 使用

```kotlin
@AutoService
class MyProcessor : AbstractProcessor() {
  ...
}
```

# Maven

```maven
<dependency>
  <groupId>com.os.auto.service</groupId>
  <artifactId>auto-service</artifactId>
  <version>0.1.1</version>
  <type>pom</type>
</dependency>
```

# Gradle

```gradle
compile 'com.os.auto.service:auto-service:0.1.1'
```
